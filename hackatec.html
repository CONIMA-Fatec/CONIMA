HackaTec
Alimentos

O objetivo principal do HackaTec Alimentos é trazer soluções para o mercado a partir das dores das empresas parceiras e apoiadoras. Para além disso, é uma ótima forma das empresas identificarem possíveis talentos, visto que a competição estimula a criatividade de desenvolvimento dos estudantes.


O HackaTec alimentos contará com certificado de participação para todos os participantes. Para a equipe vencedora, teremos premiação e homenagem!


Para além da premiação para os participantes é o momento de colocarem a mão na massa e criarem um projeto real para o portifólio.

O que é um Hackathon?

Hackathon significa maratona de programação. O termo resulta de uma combinação das palavras inglesas "hack"(programar de forma excepcional) e "marathon"(maratona).


É um evento que reúne programadores, designer e outros profissionais ligados ao desenvolvimento de software para uma maratona de programação, cujo objetivo é desenvolver um software ou solução tecnológica que atenda a um objetivo específico.
